# Описание скриптов проекта Nikita

В данном каталоге находятся служебные скрипты для сборки, установки и управления сервисом.

## Сборочные скрипты

*   `__CI.CompileAll*.cmd` - Скрипты для CI/CD пайплайнов, выполняют полную компиляцию проекта.
*   `b.2pyd.cmd` / `b.compiles2pyd.py` - Компиляция Python-модулей в бинарные `.pyd` файлы (используя Cython/PyInstaller) для защиты кода и оптимизации.
*   `c.build.*` - Различные варианты сборки (Free версия, Linux версия).
*   `c.installer.nsi` - Сценарий для NSIS (Nullsoft Scriptable Install System) для создания Windows-инсталлятора.
*   `__Clean.cmd` - Очистка директорий сборки (`build`, `dist` и временных файлов).

## Управление сервисом (Windows)

Эти скрипты используются для управления установленной службой (или для отладки дистрибутива):

*   `run.dist.install.cmd` - Установка сервиса в систему.
*   `run.dist.start.cmd` - Запуск службы.
*   `run.dist.stop.cmd` - Остановка службы.
*   `run.dist.remove.cmd` - Удаление службы из системы.
*   `run.dist.console.cmd` - Запуск скомпилированной версии в консольном режиме (для отладки).

## Утилиты

*   `add.firewall.rule.cmd` - Добавление правил в Windows Firewall для работы сервиса (открытие портов).
*   `install.python.modules.cmd` - Установка необходимых Python-библиотек из `requirements.txt` (или аналогичных).
*   `share.1cdata.sh` - Скрипт для настройки прав доступа/шаринга данных 1С (Linux).
*   `!.test.cmd` - Запуск тестов.

## Примечание по запуску

Все скрипты перемещены в папку `scripts/`. Для корректной работы они автоматически переключают рабочую директорию на корень проекта.
